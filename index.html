Sobre Mim

Aldenir Luiz é um profissional multifacetado que encontrou paixão tanto na construção física quanto na construção digital. Aos 29 anos, é casado e tem um lindo filho BPC, o que o motiva a buscar uma carreira sólida e recompensadora.

- "Atualmente, exerço as funções de pedreiro, eletricista e encanador, habilidades que aprimorei ao longo da minha trajetória em busca da estabilidade financeira para minha família. Enfrentando diariamente uma jornada repleta de desafios, me dedico a um trabalho árduo e, por vezes, insalubre. Essas experiências moldaram minha resiliência e determinação, impulsionando-me na construção de um futuro sólido para aqueles que amo."

Além de suas habilidades na construção, Aldenir nutre um interesse profundo por eletrônica e mecatrônica. Esses hobbies revelam sua curiosidade e fascínio pelo mundo da tecnologia e automação. Mesmo enquanto realiza trabalhos físicos, ele encontra tempo para explorar sua paixão pelo desenvolvimento de dispositivos eletrônicos. O sonho de Aldenir é transicionar para a área de desenvolvimento de software profissional. Essa ambição reflete seu desejo de expandir suas habilidades para além das atividades físicas e ingressar em um campo que combina sua paixão por tecnologia com oportunidades de crescimento profissional. Seu perfil diversificado e conjunto único de habilidades contribuem para uma narrativa única e inspiradora, destacando a ideia de que é possível buscar novos horizontes, independentemente da trajetória inicial.

Sistema de gerenciamento de dados de crediario Corro Variedades

O projeto consiste em um sistema desktop para o gerenciamento de dados relacionados ao crediário da loja "Corró Variedades". A aplicação utiliza a biblioteca Tkinter para criar a interface gráfica e SQLite como banco de dados para armazenar informações.

Este sistema possui funcionalidades básicas para o uso diario na empresa, como adicionar, visualizar, editar e excluir dados relacionados a rotas de crediário. Além disso, oferece a opção de imprimir relatórios em PDF. Todo o código faz uso de orientação a objetos para organizar a lógica do programa de maneira modular e reutilizável tornando a sua manutencao e escalabilidade muito mais eficiente.

A estrutura principal do código está dividida em várias classes entre elas estão:

Janela Principal: Esta Janela que é responsável por criar a interface gráfica principal do sistema. A janela principal inclui funcionalidades para visualizar dados, cadastrar rotas, editar informações, e imprimir relatórios em PDF. O código utiliza diversos widgets, como botões, entradas, comboboxes, e treeviews para exibir dados em forma de tabela.

Classe Layout: A classe Layout é responsável por gerar dinamicamente a interface gráfica do sistema de planilhas. Ela define métodos para criar diferentes tipos de widgets, como entradas e rótulos, e organizar esses widgets em cartões (cards) para uma melhor apresentação. O código também inclui métodos para criar widgets adicionais, como tarjas de descrição e subwidgets.

Gerenciamento de dados: A classe HandlerDB, que é responsável por lidar com o banco de dados SQLite. A classe inclui métodos para executar consultas SQL, adicionar dados, recuperar dados, e verificar a existência de tabelas. O código também trata erros de conexão com o banco de dados.

Observação: O código fornecido é extenso, e esta descrição destaca os aspectos principais. Caso tenha alguma dúvida específica ou precise de mais informações sobre alguma parte do código, sinta-se à vontade para perguntar, ou consulte o repositorio em: Financeiro crediario no GitHub
